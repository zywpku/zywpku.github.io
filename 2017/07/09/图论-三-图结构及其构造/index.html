Set System and Hypergraphs

$-\cdot$$Set$ $System$: An ordered pair $(V,\mathcal{F})$, where $V$ is a set of elements and $\mathcal{F}$ is a family of subsets of $V$。 其中$V$是元素集，$\mathcal{F}$是V的子集构成的集合。The set system $(V,\mathcal{F})$ is a loopless graph（无环图）。
$-\cdot$$Hypergraphs（超图）$: The set system can be thought of as generalizations of graphs, and are usually referred to as hypergraphs. The elements of $V$ are then called the vertices of the hypergraph, and the elements of $\mathcal{F}$ its edges or hyperedges.

Incidence Graphs

$-\cdot$$Incidence$ $Graphs$: A natural graph associated with a set system $H = (V, \mathcal{F})$ is the bipartite graph $G[V,\mathcal{F}]$, where $v\in V$ and $F \in \mathcal{F}$ are adjacent if $v \in F$. 可以将其理解为：$V$，single 图的点集； F，single 图的路径。如下图所示。

Intersection Graphs

With each set system $(V, \mathcal{F} )$ one may associate its intersection graph. This is the graph whose vertex set is $\mathcal{F}$ , two sets in $\mathcal{F}$ being adjacent if their intersection is nonempty. For historical reasons, this graph is known as the line graph of $G$ and denoted $L(G)$.

When $V = R$ and $\mathcal{F}$ is a set of closed intervals of $R$, the intersection graph of $(V, \mathcal{F})$ is called an interval graph.
